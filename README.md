# metabarpipe

An organized pipeline for Windows that leverage on vsearch and cutadapt

# Installation

```R
# Install the remote package if missing
if (!requireNamespace(remotes, quietly = TRUE)) {
  install.packages(remotes)
}

# install the metabarpipe package
remotes::install_github("alexology/metabarpipe", ref = "main")
```
